Lista de tarefas

Projeto desenvolvido em Python com o objetivo de praticar lógica de programação,
funções, módulos e  manipulação de listas.

Sistema de tarefas que permite o cadastro, conclusão e remoção de tarefas atráves
do menu interativo no terminal.

Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Menu interativo no terminal

Estrutura do Projeto

- sistema.py - Controle principal do programa
- lib/interface.py - interface e menu do sistema
- lib/tarefas.py - Manipulação das tarefas

Tecnologias Utilizadas

- Python 3

Como executar

1. Baixe os arquivos do projeto
2. Execute o arquivo sistema.py
